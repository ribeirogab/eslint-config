# @ribeirogab/eslint-config

## Install

To install free-translate, you can use NPM:

```zsh
npm install @ribeirogab/npm-config
```

## Setup

Create `.eslintrc` file and extends `@ribeirogab/npm-config`

```json
{
  "extends": [
    "@ribeirogab/eslint-config"
  ]
}
```
